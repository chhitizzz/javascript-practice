------- JavaScript Questions -------

1. Function that takes two numbers as input and returns their sum.
2. Function that takes a string as input and returns its length.
3. Function that takes an array of numbers as input and returns the sum of all the numbers in the array.
4. Function that takes a number as input and checks if it is positive, negative, or zero. Return "Positive," "Negative," or "Zero" accordingly.
5. Function that takes two numbers as input and returns the larger of the two.
6. Function that takes a string as input and checks if it contains the word "JavaScript". Return true if it does, false otherwise.
7. Function that takes a number as input and checks if it is divisible by both 3 and 5. Return true if it is, false otherwise.
8. Function that takes a number as input and checks if it is even. If it's even, return "Even," otherwise return "Odd".
9. Function that takes two strings as input and checks if they are equal. Return true if they are equal, false otherwise.
10. Function that takes a number as input and returns "Fizz" if the number is divisible by 3, "Buzz" if it's divisible by 5, and "FizzBuzz" if it's divisible by both 3 and 5. Otherwise, return the number itself.
11. Function that takes a year as input and checks if it's a leap year. Return true if it's a leap year, false otherwise.
12. Function that takes an array of numbers as input and returns the number of positive, negative, and zero values in the array as an object.
13. Function that takes three numbers as input and returns the largest of the three without using the built-in `Math.max()` method.
14. Program that prints numbers from 1 to 10.
15. Program that prints even numbers from 1 to 20.
16. Function that takes a number as input and prints a multiplication table for that number up to 10.
17. Function that takes an array of numbers as input and calculates the sum of all the numbers using a loop.
18. Program that iterates over an array of strings and prints each string to the console.
19. Function that takes a number as input and checks if it's a prime number using a loop. Return true if it's prime, false otherwise.
20. Program that iterates over an array of objects representing people (with name and age properties) and prints their names to the console.
21. Function that takes a string as input and counts the number of occurrences of a specific character using a loop.
22. Program that iterates over an array of numbers and returns a new array with each number squared.
23. Function that takes a positive integer as input and prints a right-angled triangle of stars (*) with the given height.
24. Function that takes an array of numbers as input and returns the product of all the numbers.
25. Function that takes an array of strings as input and returns a new array with all the strings in uppercase.
